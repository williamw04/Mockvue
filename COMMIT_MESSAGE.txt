Add LangGraph agentic system integration demo

This commit adds a complete demonstration of integrating an agentic AI system
with the document service using custom BlockNote blocks. The implementation
includes:

FEATURES:
- Custom BlockNote blocks: Question ❓, Response 💬, Context 📋, Notes 📝
- Three AI features: Brainstorm 💡, Feedback 📝, Draft ✍️
- Mock agentic service layer (web & electron)
- AI Assistant panel with feature selection
- Response selector for choosing content to analyze
- Enhanced document page with custom block insertion
- Dashboard banner promoting AI demo

ARCHITECTURE:
- IAgenticService interface following app's service pattern
- WebAgenticService & ElectronAgenticService implementations
- useAgentic() React hook for component access
- Custom BlockNote schema extending default blocks
- Type-safe implementation with TypeScript

COMPONENTS:
- AIAssistantPanel: Main AI features UI with processing indicators
- ResponseSelector: Interactive response management and selection
- DocumentPageWithAI: Enhanced editor with custom blocks and AI panel
- Updated Dashboard: Prominent AI demo banner and navigation

AI FEATURES:
1. Brainstorm: Generate creative ideas from document context
2. Feedback: Provide detailed analysis of selected responses
3. Draft: Create structured documents from responses and context

MOCK IMPLEMENTATION:
- Simulates AI processing with realistic delays (1.5-2.5s)
- Pre-generated varied responses for consistent demo experience
- No API keys or external services required
- Architecture ready for real LangGraph integration
